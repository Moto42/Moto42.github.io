YouTube Soundboard

A 'soundboard' built in React.js, that plays videos from YoutTube.
When a button is pressed, the selected video is played. Users can add a new video to the list, and remove existing buttons. A user's customized list is saved to their browswer's local storage, allowing the user to keep a customized list between sessions.
